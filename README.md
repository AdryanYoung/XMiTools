# Mi Tweaks - MIUI 时钟模块
An xposed module for MIUI10 tweaks. / MIUI10 拓展模块

# 注意
- 仅适用于MIUI 10，其他版本的 MIUI 系统以及非 MIUI 系统，请慎用。
- 支持 Xposed，EdXposed 和 太极阳。

# 功能
- 状态栏
  1. 状态栏显示秒数
  2. 状态栏时间对齐方式（居左，居中，居右）
  3. 自定义状态栏时间格式
- 下拉状态栏
  1. 时间显示秒数
- 锁屏界面
  1. 水平时钟显示秒数
  2. 垂直时钟显示秒数
